FountainAI Character Management API (2.0.0)

This API handles character management, including actions, spoken words, and their paraphrases.

Retrieve All Characters

Lists all characters stored within the application, offering an overview of the characters available for screenplay development. This endpoint leverages Redis caching to improve query performance.

BearerAuth

Responses

Create a New Paraphrase for a Character

Allows for the creation of a new paraphrase linked to a character. Clients must provide the paraphrased text and a commentary explaining the link to the original character. RedisAI integration provides additional validation.

Authorizations:
BearerAuth
path Parameters
characterId
required
integer

The unique identifier of the character to which the paraphrase will be linked.

Request Body schema: application/json
required

A JSON object containing the new paraphrase's details and its link commentary.

paraphraseId
integer <int64>

The unique identifier for the Paraphrase, automatically generated upon creation.

originalId
required
integer

The ID of the original action to which this paraphrase is linked.

text
required
string

The paraphrased text of the original action.

commentary
required
string

An explanatory note on why the paraphrase is linked to the original action.

Create a New Action

Allows for the creation of a new action entity. Clients must provide an action description and its sequence within a script. This endpoint integrates with RedisAI to provide recommendations or validations based on predefined models.

Authorizations:
BearerAuth
Request Body schema: application/json
required

A JSON object containing the new action's details.

actionId
integer <int64>

The unique identifier for the Action, automatically generated upon creation.

description
required
string

A textual description outlining what happens in this action.

sequence
required
integer <int32>

The numerical order of the action within its script, used to organize actions sequentially.
