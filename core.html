FountainAI Core Script Management API (2.0.0)

Download OpenAPI specification:Download

This API handles script management, including section headings, transitions, and orchestration-related functionalities.

Retrieve All Scripts

Lists all screenplay scripts stored within the system. This endpoint leverages Redis caching for improved query performance.

Authorizations:
BearerAuth

Responses

Production server for Core Script Management API

https://core.fountain.coach/scripts

Development server (Docker environment)

http://localhost:8080/scripts

Response samples

  • 200
Content type
application/json
[
  • {
    • "scriptId": 1,
    • "title": "Sunset Boulevard",
    • "description": "A screenplay about Hollywood and faded glory.",
    • "author": "Billy Wilder",
    • "sequence": 1
    }
]

Create Section Heading

Creates a new Section Heading within a script, specifying its sequence, title, and associated script ID. RedisAI middleware provides recommendations and validation during the creation process.

Authorizations:
BearerAuth
Request Body schema: application/json
required

Data required to create a new Section Heading.

headingId
integer

Unique identifier for the Section Heading.

scriptId
required
integer

Identifier of the script this Section Heading belongs to.

title
required
string

Title of the Section Heading.

sequence
required
integer

Order sequence of the Section Heading within the script.
